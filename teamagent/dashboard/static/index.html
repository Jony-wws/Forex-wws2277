🧠 ГЛАВНЫЙ СТРАТЕГИЧЕСКИЙ АГЕНТ …

Тактический мета-агент. Каждые 5 часов прогоняет 28 пар × 4 сессии × 120 вариантов на 4 окнах (3д / 5д / 7д / 10д) свежих данных Yahoo и подмешивает ансамбль из 8 реальных источников (COT · FRED · Regime · Radar · Multi-TF · Momentum · News · Specialist). Для каждой ячейки выбирает лучшее окно + лучший variant и присваивает статус КАЧЕСТВЕННАЯ (WR ≥ 70% + Wilson_lower ≥ 60%) / ВЕРОЯТНАЯ (55–70%) / ЗАМОРОЖЕНА (нет edge).
Качественные ячейки добавляют ±3 score-голоса в forecast_scanner-е, Вероятные — ±2. 365-дневный locked-baseline остаётся эталоном — мета-агент его не перезаписывает.
Только реальные данные: Yahoo Finance · CFTC COT · FRED · 64 живых субагента. Без симуляторов и mock-данных.

🛡️ ГАРАНТИИ СТАБИЛЬНОСТИ мат. границы · 95% доверие

Все цифры — это математические нижние границы по реальным историческим данным, не прогноз и не теория. Wilson 95% CI = «худший правдоподобный WR», bootstrap p5 = «худший ожидаемый PnL/сделку», VaR/CVaR = «худшие 5% реальных потерь», conformal coverage = «90% будущих цен попадут в этот коридор». Никаких симуляторов, никакой случайности — bootstrap идёт по реальным закрытым сделкам с фиксированным seed.

  • PROGNOZY-28 — единая таблица прогнозов по 28 парам. Каждые 5 минут scanner пересчитывает score (−44..+44), вероятность (50–92%), за/против. Кликни строку — увидишь breakdown агентов и Стакан.

  • 60+ агентов — каждый отдельный процесс с heartbeat. Watchdog убивает зависших, orchestrator перезапускает.

📊 COT positioning (CFTC · 7 валют · 52 недели · z-score контраприан)

Реальные публичные данные CFTC: позиции hedge funds (leveraged money) во фьючерсах EUR/GBP/JPY/CHF/AUD/CAD/NZD на CME. Когда позиции растянуты (|z| > 1.5 за 52 недели) — высокая вероятность mean-reversion (контрапариан). Это «как смотрят macro hedge funds»: тот самый отчёт который выходит каждую пятницу. Источник: CFTC public Socrata API.
