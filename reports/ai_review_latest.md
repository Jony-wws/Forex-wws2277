# 🤖 AI-обзор стратегии — авто-генерация

_Этот файл создаётся GitHub Actions workflow `ai_review.yml` каждый раз после завершения 5-часового цикла._

## 🧠 AI-обзор (GitHub Models · openai/gpt-4o-mini · бесплатно)

# Диагноз
1. **Низкая стабильность сигналов**: В текущем цикле не удалось достичь цели по количеству стабильных пар с винрейтом ≥70%. Это указывает на недостаточную фильтрацию сигналов, что приводит к большому количеству неэффективных сделок.
2. **Слабые тренды**: Из 28 валютных пар только 3 показывают сильные тренды, что не соответствует требованию о наличии минимум 3 сильных трендов каждые 5 часов.
3. **Низкий винрейт**: Большинство пар имеют винрейт ниже 60%, что говорит о том, что текущие параметры не позволяют отсеивать слабые сигналы.

# Рекомендации
1. **Увеличить STRONG_CONFIDENCE**: Установить значение на уровне **92%** (вместо 88%). Это позволит отсеивать менее уверенные сигналы.
2. **Увеличить STRONG_RATIO**: Установить значение на уровне **0.65** (вместо 0.55). Это повысит требование к количеству успешных сигналов по сравнению с неуспешными.
3. **Увеличить STRONG_ADX_H1**: Установить значение на уровне **30** (вместо 25). Это позволит фильтровать только те тренды, которые действительно сильные.
4. **Увеличить STRONG_ADX_H4**: Установить значение на уровне **25** (вместо 20). Это также повысит качество отбираемых сигналов.
5. **Увеличить STRONG_PERSISTENCE**: Установить значение на уровне **90%** (вместо 80%). Это обеспечит более высокую вероятность того, что тренд будет устойчивым.

# Что НЕ менять
- **Параметры ADX для других таймфреймов**: Не менять параметры ADX для таймфреймов M15 и D1, так как они уже обеспечивают необходимую гибкость для анализа.
- **Стратегии и подходы**: Не менять основные стратегии (тренд, откат), так как они уже показали свою эффективность в определённых условиях.

# Риски
1. **Снижение количества сигналов**: Ужесточение фильтров может привести к значительному снижению количества доступных сигналов, что может негативно сказаться на общей активности торговли.
2. **Пропуск потенциально прибыльных сделок**: Более строгие параметры могут отсеивать не только слабые, но и некоторые сильные сигналы, что может привести к упущенным возможностям.
3. **Необходимость адаптации**: Изменение параметров потребует времени для адаптации и тестирования, что может временно снизить эффективность системы.

## 🧠 Эвристический AI-обзор (без LLM)

### Метрики
- **WR за последние 5 часов:** —% (0 решений)
- **WR на 28-парном бэктесте:** —%
- **Деградировавшие пары:** EURCHF

### Диагноз
- WR в норме — текущая жёсткость работает.
- Пары в деградации (1): `EURCHF` — рассмотреть исключение из топа на 24ч.

### Рекомендации к параметрам
- В `app/cycle.py._select_strict` добавить временный blacklist для деградировавших пар: ['EURCHF'].

### Точки правки
