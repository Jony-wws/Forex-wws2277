# TradingView Pine Script — FOREX-28

Два скрипта для TradingView, которые повторяют логику нашего Python-бекенда
(`app/analyzer.py`):

| Файл | Что делает |
|---|---|
| `forex_signals_indicator.pine` | **Индикатор**: метки BUY / SELL на графике + таблица с уверенностью %. Без бэктеста. Для повседневного использования. |
| `forex_signals_strategy.pine`  | **Стратегия**: тот же сигнал + правила входа/выхода (бинарный 0.10 % выигрыш/проигрыш на 5 ч / 24 ч). Запускается через TradingView Strategy Tester и показывает win-rate, profit factor, drawdown за 1–5 лет истории. |

## Логика (одинакова в обоих)

* Голосование 11 блоков на 4 таймфреймах: **D1 + H4 + H1 + M15**.
* Индикаторы: EMA50/200 trend, RSI, MACD, Bollinger %B, Stochastic, ADX (+DI/-DI),
  Williams %R, momentum, multi-TF agreement, EMA20/50 cross на M15.
* Сумма голосов → `score / max_possible`.
* Уверенность = `50 + 45 · (1 − exp(−3.66 · score/max))` → диапазон 50–95 %.
* Сигнал отображается / открывается **только** при уверенности ≥ 80 % (можно
  поменять в инпутах).
* Стратегия закрывает сделку по первому из:
  * `+0.10 %` от точки входа → **выигрыш**
  * `−0.10 %` от точки входа → **проигрыш**
  * `Horizon` часов истекли (по умолчанию 5 ч) → закрытие по рынку
  * Это бинарная логика, такая же как в `app/cycle.py`.

## Как использовать в TradingView

1. Открыть TradingView, открыть нужную пару (например `OANDA:EURUSD`).
2. Нажать **Pine Editor** (внизу экрана).
3. Скопировать содержимое `.pine` файла и вставить в редактор.
4. Нажать **Save**, ввести имя индикатора.
5. Нажать **Add to chart** — скрипт появится на графике.
6. Для стратегии: после добавления откроется вкладка **Strategy Tester** —
   там видно `Net Profit`, `Total Trades`, `Win Rate`, `Profit Factor`, `Max Drawdown`.

## Как итерировать качество

После просмотра бэктеста:
* Если win-rate низкий — поднять `gateConf` до 85 / 90.
* Если сделок очень мало — снизить `gateConf` до 75 (риск выше).
* `winPct` (порог 0.10 %) можно увеличить до 0.20 %, если хотите более крупные движения и согласны на меньше срабатываний.
* `horizonH` — 5 ч для бинарных опционов, 24 ч для свинг-трейда.

Меняйте параметры прямо в окне настроек индикатора в TradingView, никакого кода трогать не нужно.
