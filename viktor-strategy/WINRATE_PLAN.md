## Честная отправная точка (зафиксировано бэктестами, см. README §2)

- База угадывания направления EUR/USD на 5ч ≈ 50%. Безубыток при выплате 80% = 55.6%.
- **Гарантированных 70% каждый месяц не существует** — это проверено на 35 месяцах
  реальных данных (0 из 35 месяцев у мульти-индикаторных систем).
- Единственный путь ВВЕРХ по winrate — **селективность**: торговать реже, но только
  моменты, где сразу несколько НЕЗАВИСИМЫХ источников показывают одно направление.
- Поэтому план = не «новый индикатор», а **фильтр A+ сетапов** + новые независимые
  источники информации. Каждая добавленная независимая «пара глаз» с эджем >50%
  поднимает вероятность совпадения общего вердикта с рынком.

## Что внедрено (13.06.2026)

### 1. Режим «ТОЛЬКО ВХОД» (signal mode)
Сканер работает каждые 30 минут, но пишет ТОЛЬКО когда сетап A+:
- момент входа 🟢 (вход на откате, не вдогонку), и
- минимум **2 независимые группы подтверждений** из пяти:
  1. КРУПНЫЕ ИГРОКИ (smart_money.py — структура, дисплейсмент, OB, FVG, пулы)
  2. ТРЕНД (EMA50/200 D1/H4/H1, |score| ≥ 0.8)
  3. CONFLUENCE+ (новый модуль, |балл| ≥ 1.5 — см. ниже)
  4. Свежий свип ликвидности В НАШУ сторону
  5. RSI-ЭКСТРЕМУМ — наш единственный бэктест-подтверждённый эдж (54-58%)
- блокеры: флэт без вердикта крупных, свип ПРОТИВ, 3+ confluence-фактора против,
  красная новость в окне 5ч (жёсткий стоп).

**Каждое присланное сообщение = вход СРАЗУ.** «Не заходить» больше не приходит —
плохие моменты система отсеивает молча.

### 2. Новые «планы» — модуль `confluence_plus.py`
Аналогичные «крупным игрокам» независимые источники:

| План | Что смотрит | Почему работает |
|---|---|---|
| **COT (CFTC)** | РЕАЛЬНЫЕ позиции крупных спекулянтов во фьючерсе EUR (публичный отчёт биржи, еженедельно) | Единственные данные, где видно, куда крупные УЖЕ поставили деньги, а не «следы на графике» |
| **DXY** | Индекс доллара: тренд + дивергенции с EUR/USD | EUR/USD — это евро ПРОТИВ доллара; если доллар слабеет везде, евро растёт |
| **US10Y** | Доходность 10-летних облигаций США | Деньги текут туда, где платят больше: рост доходности → сильный доллар |
| **RSI-экстремум** | H1 RSI ≤25 / ≥75 | Наш лучший проверенный эдж: 54-58% на 17 000 баров |
| **Круглые уровни** | Отбои/стены на 00/50 | Психология толпы + опционные барьеры маркетмейкеров |
| **Свечи** | Поглощение/пин-бар H1 по направлению | Подтверждение, что в моменте побеждает наша сторона |

## Что дальше (по мере накопления статистики)

1. **Журнал сделок** (≥100 сделок) → реальный winrate по типам сетапов.
2. **Тюнинг весов**: отключаем группы, которые на ЖИВЫХ сделках не дают эджа,
   усиливаем те, что дают. Это единственный честный способ приближаться к 70%.
3. Кандидаты на добавление: спред US10Y−Bund, объёмы фьючерса 6E, сезонность
   по дням недели/месяца.

## Правила, которые важнее любого индикатора

- Риск ≤1-2% депозита на сделку — ВСЕГДА (одна серия неудач не должна убивать счёт).
- Не торговать против новостного фильтра и против свежего свипа — никогда.
- Меньше сделок = выше качество. Пропущенный день без сигнала — это система
  РАБОТАЕТ, а не молчит.
- Оценка системы — только на 100+ сделках. 5-10 сделок ничего не доказывают.
